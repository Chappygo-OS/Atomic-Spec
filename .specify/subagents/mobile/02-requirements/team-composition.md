## Team Structures

### MVP Team (2-4 people)
```
┌─────────────────────────────────────┐
│        Product/Founder (1)          │
│    Vision, PM, stakeholder mgmt     │
├─────────────────────────────────────┤
│      Developer(s) (1-2)             │
│   Cross-platform or single platform │
├─────────────────────────────────────┤
│      Designer (0.5-1)               │
│   UX/UI, freelance or part-time     │
└─────────────────────────────────────┘

Total: 2-4 people
Backend: BaaS or simple API
QA: Developer-shared
DevOps: Developer-shared
```

### Growth Team (5-10 people)
```
┌─────────────────────────────────────┐
│          Product Manager (1)         │
├─────────────────────────────────────┤
│          Design (1-2)                │
│       UX Designer + UI Designer      │
├─────────────────────────────────────┤
│       Mobile Development (2-4)       │
│   iOS (1-2) + Android (1-2)         │
│   OR Cross-platform (2-3)           │
├─────────────────────────────────────┤
│       Backend Development (1-2)      │
├─────────────────────────────────────┤
│            QA (1)                    │
└─────────────────────────────────────┘

Total: 6-10 people
DevOps: Shared with developers
```

### Scale Team (15-30 people)
```
┌─────────────────────────────────────┐
│    Product (2-3)                     │
│  PM + Associate PM + Analyst         │
├─────────────────────────────────────┤
│    Design (3-4)                      │
│  UX Lead + UI + Motion + Research    │
├─────────────────────────────────────┤
│    iOS Team (4-6)                    │
│  Tech Lead + Senior + Mid + Junior   │
├─────────────────────────────────────┤
│    Android Team (4-6)                │
│  Tech Lead + Senior + Mid + Junior   │
├─────────────────────────────────────┤
│    Backend Team (3-5)                │
│  Lead + Seniors + Mid                │
├─────────────────────────────────────┤
│    QA Team (3-4)                     │
│  Lead + Manual + Automation          │
├─────────────────────────────────────┤
│    DevOps/SRE (1-2)                  │
└─────────────────────────────────────┘

Total: 20-30 people
```
