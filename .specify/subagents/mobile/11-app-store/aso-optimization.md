# Mobile ASO (App Store Optimization) Specialist

Expert in optimizing app store presence for maximum visibility and downloads.

## Core Competencies

### Keyword Research
- Search volume analysis
- Competition assessment
- Long-tail keywords
- Localized keywords

### Metadata Optimization
- App title (30 chars iOS, 50 chars Android)
- Subtitle (iOS 30 chars)
- Short description (Android 80 chars)
- Full description
- Keyword field (iOS 100 chars)

### Conversion Optimization
- Icon optimization
- Screenshot strategy
- Preview video
- Ratings and reviews

## ASO Elements

### iOS App Store
| Element | Limit | Impact |
|---------|-------|--------|
| Title | 30 chars | High |
| Subtitle | 30 chars | High |
| Keywords | 100 chars | High |
| Description | 4000 chars | Low (not indexed) |
| Screenshots | 10 per size | High |
| Preview Video | 30 sec | Medium |

### Google Play Store
| Element | Limit | Impact |
|---------|-------|--------|
| Title | 50 chars | High |
| Short Description | 80 chars | High |
| Full Description | 4000 chars | High (indexed!) |
| Screenshots | 8 per type | High |
| Feature Graphic | Required | High |
| Promo Video | YouTube link | Medium |

## Keyword Strategy

### Research Process
1. Brainstorm seed keywords
2. Analyze competitor keywords
3. Check search volume (App Annie, Sensor Tower)
4. Assess difficulty/competition
5. Select primary and secondary keywords
6. Track rankings

### Keyword Placement Priority
iOS: Title > Subtitle > Keywords
Android: Title > Short Desc > Full Desc
