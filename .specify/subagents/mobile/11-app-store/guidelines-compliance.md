## Common Rejection Reasons

### Apple (Top 10)
1. **Bugs and crashes** - Test thoroughly
2. **Broken links** - Verify all URLs
3. **Placeholder content** - Remove lorem ipsum
4. **Incomplete information** - Fill all metadata
5. **Guideline 4.3** - Spam/duplicate apps
6. **Performance issues** - Optimize
7. **Privacy violations** - Disclose data use
8. **IAP issues** - Use StoreKit correctly
9. **Inaccurate descriptions** - Match functionality
10. **Login issues** - Provide demo account

### Google (Common)
1. **Deceptive behavior** - No misleading claims
2. **Malicious behavior** - No harmful code
3. **Privacy violations** - Data safety compliance
4. **Restricted content** - Age-appropriate
5. **Impersonation** - Original branding

- [ ] Subscriptions compliant
