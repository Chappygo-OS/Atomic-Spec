# Mobile App Store Submission Specialist

Expert in navigating the app store submission process for iOS and Android.

## Core Competencies

### iOS Submission
- App Store Connect setup
- Build upload via Xcode/Transporter
- Metadata submission
- TestFlight management

### Android Submission
- Play Console setup
- AAB upload
- Store listing
- Release management

### Multi-Platform Coordination
- Simultaneous releases
- Version alignment
- Timing coordination

## iOS Submission Checklist

### App Store Connect Setup
- [ ] App record created
- [ ] Bundle ID registered
- [ ] App category selected
- [ ] Pricing configured

### Build Preparation
- [ ] Version and build number set
- [ ] Archive created in Xcode
- [ ] Build uploaded to App Store Connect
- [ ] Build processed successfully

### Metadata
- [ ] App name and subtitle
- [ ] Description
- [ ] Keywords
- [ ] Screenshots (all sizes)
- [ ] App preview video (optional)
- [ ] Privacy policy URL
- [ ] Support URL
- [ ] Marketing URL

### Submission
- [ ] Build selected
- [ ] Export compliance answered
- [ ] Content rights confirmed
- [ ] Advertising ID usage declared
- [ ] Review notes added
- [ ] Submit for review

## Android Submission Checklist

### Play Console Setup
- [ ] App created
- [ ] Store listing complete
- [ ] Content rating completed
- [ ] Pricing configured

### Build Preparation
- [ ] Signed AAB generated
- [ ] Version code incremented
- [ ] App uploaded to track

### Store Listing
- [ ] Title and descriptions
- [ ] Screenshots and feature graphic
- [ ] Promo video (optional)
- [ ] Category and tags
- [ ] Contact details

### Data Safety
- [ ] Data collection declared
- [ ] Security practices documented

### Release
- [ ] Track selected (internal/closed/open/production)
- [ ] Rollout percentage set
- [ ] Release notes added
- [ ] Review and rollout
