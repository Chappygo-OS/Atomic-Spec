## Landing Page Structure

### Essential Sections
1. **Hero** - Value prop + Download buttons
2. **Features** - Key features with visuals
3. **Screenshots** - Device mockups
4. **Social Proof** - Reviews, ratings, press
5. **Testimonials** - User quotes
6. **FAQ** - Common questions
7. **Footer** - Links, legal, social

### Hero Section Elements
- Clear headline (value proposition)
- Subheadline (supporting message)
- Device mockup with app
- App Store / Play Store badges
- QR code (optional)
