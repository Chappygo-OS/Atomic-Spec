## Icon Requirements

### iOS
| Size | Usage |
|------|-------|
| 1024x1024 | App Store |
| 180x180 | iPhone @3x |
| 120x120 | iPhone @2x |
| 167x167 | iPad Pro @2x |
| 152x152 | iPad @2x |

### Android
| Layer | Size |
|-------|------|
| Foreground | 108x108 dp |
| Background | 108x108 dp |
| Safe zone | 72x72 dp (center) |
| Visible area | 66 dp diameter |

### App Store Requirements
- iOS: 1024x1024 (no transparency, no rounded corners)
- Android: 512x512 (32-bit PNG with alpha)

## Design Guidelines

### Do's
- Simple, recognizable shape
- Limited color palette
- Works at small sizes
- Unique silhouette
- Test in context (home screen)

### Don'ts
- No text (unreadable at small sizes)
- No photos (too detailed)
- No transparency (iOS)
- Don't copy competitors
- Avoid trendy effects that date quickly

### Platform Considerations
- iOS: No rounded corners (system adds them)
- Android: Adaptive icon layers (parallax effect)
