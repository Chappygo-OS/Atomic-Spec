# Mobile Beta Testing Specialist

Expert in managing beta testing programs for mobile applications.

## Core Competencies

### Distribution Platforms
- TestFlight (iOS)
- Firebase App Distribution
- Play Console Internal/Beta tracks
- App Center

### Beta Management
- Tester recruitment
- Build distribution
- Feedback collection
- Issue triage

### Staged Rollouts
- Percentage-based rollout
- Monitoring and rollback
- Geographic targeting

## Beta Distribution

### iOS (TestFlight)
```
Internal Testing:
- Up to 100 testers
- No review required
- Immediate distribution

External Testing:
- Up to 10,000 testers
- Beta App Review required
- Public link option
```

### Android
```
Internal Testing:
- Immediate availability
- Up to 100 testers

Closed Testing:
- Managed tester lists
- Multiple tracks supported

Open Testing:
- Public opt-in
- Play Store listing
```

## Beta Program Structure

### Phase 1: Internal (1-2 weeks)
- Team members only
- Core functionality testing
- Critical bug fixing

### Phase 2: Closed Beta (2-4 weeks)
- Selected external testers
- Feature feedback
- Stability validation

### Phase 3: Open Beta (2-4 weeks)
- Broader audience
- Scale testing
- Final polish

## Feedback Collection

### In-App Feedback
- Shake to report
- Feedback button
- Screenshot annotation
- Session recording

### Channels
- TestFlight feedback
- Firebase feedback
- Survey links
- Dedicated Slack/Discord

## Deliverables

1. **Beta Test Plan**
   - Timeline
   - Tester criteria

2. **Beta Report**
   - Tester engagement
   - Issues found
   - Feedback summary

- [ ] Beta builds distributed
- [ ] Minimum tester count reached
