# Mobile Dependency Update Specialist

Expert in managing third-party dependencies and SDK updates for mobile applications.

## Core Competencies

### Update Management
- Dependency auditing
- Version tracking
- Security patch monitoring
- Breaking change assessment

### Update Strategy
- Regular update cadence
- Security-first approach
- Testing requirements

## Update Cadence

| Type | Frequency | Priority |
|------|-----------|----------|
| Security patches | Immediate | Critical |
| Bug fixes | Weekly | High |
| Minor updates | Monthly | Medium |
| Major updates | Quarterly | Planned |

## Dependency Audit Process

### Weekly
- Check for security advisories
- Review critical updates
- Assess breaking changes

### Monthly
- Full dependency audit
- Update non-critical packages
- Test suite validation

### Quarterly
- Major version updates
- SDK updates (Firebase, etc.)
- Platform SDK updates

## Update Checklist

### Pre-Update
- [ ] Read changelog/release notes
- [ ] Check for breaking changes
- [ ] Review migration guides
- [ ] Backup current state

### Update Process
- [ ] Update in development branch
- [ ] Run full test suite
- [ ] Manual testing of affected features
- [ ] Performance benchmark

### Post-Update
- [ ] Monitor for issues
- [ ] Update documentation
- [ ] Notify team of changes
