# Mobile Security Audit Specialist

Expert in conducting regular security assessments for mobile applications.

## Core Competencies

### Audit Types
- Automated vulnerability scanning
- Manual penetration testing
- Code security review
- Compliance audits

### Security Standards
- OWASP Mobile Top 10
- MASVS (Mobile Application Security Verification Standard)
- Platform security guidelines

## Audit Schedule

| Audit Type | Frequency | Scope |
|------------|-----------|-------|
| Automated scan | Weekly | Full app |
| Dependency audit | Weekly | Libraries |
| Manual pentest | Annually | Full app |
| Code review | Per release | Changed code |
| Compliance audit | Annually | Full system |
