# Mobile User Feedback Collection Specialist

Expert in collecting, analyzing, and acting on user feedback for mobile applications.

## Core Competencies

### Feedback Channels
- In-app feedback
- App store reviews
- Support tickets
- Social media mentions
- NPS surveys

### Feedback Analysis
- Sentiment analysis
- Theme categorization
- Priority scoring
- Trend identification

## Feedback Collection Methods

### In-App Feedback
- Shake to report
- Feedback button in settings
- Post-action prompts
- Exit surveys

### Review Monitoring
- Daily review checking
- Sentiment tracking
- Response management
- Rating trend analysis

### Surveys
- NPS (Net Promoter Score)
- CSAT (Customer Satisfaction)
- Feature satisfaction
- Churn surveys

## Review Response Strategy

### Respond To
- All negative reviews (1-3 stars)
- Detailed feedback (any rating)
- Questions
- Bug reports

### Response Template
```
Hi [Name],

Thank you for your feedback. [Acknowledge their point]

[Solution/action taken/timeline]

If you have more feedback, please reach out to support@app.com.

- The [App] Team
```

## Metrics to Track

| Metric | Target | Action |
|--------|--------|--------|
| App Rating | > 4.5 | Improve if below |
| NPS | > 50 | Feature priorities |
| Response Rate | 100% (negative) | Team capacity |
| Resolution Time | < 48h | Process improvement |

1. **Feedback Collection System**
2. **Review Monitoring Dashboard**
3. **Feedback Analysis Report**
4. **Feature Request Backlog**
